# K-Nel-M1721
* 本仓库为K-Nel的魅蓝Note6版仓库
* 该仓库仅存储为魅蓝Note6编译的K-Nel内核
* 编译者：科家（KNKJ55）
* 编译使用设备：魅蓝Note6
* 编译使用系统：Chroot - Ubuntu 22.04 LTS
* 测试通过机型：魅蓝Note6
* 目标机型：魅蓝Note6
* 初始源码来源：@Bilibili极吧弯
* 已实现功能：
* 1.KernelSU
* 2.Lxc-Docker
* 3.Kernel Virtual Machine
# 注意！
# 刷机有风险，搞机需谨慎！
